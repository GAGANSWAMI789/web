Use of pseudo-class and pseudocode-elements
In CSS, a pseudo-class selector targets elements depending on their state rather than on information from the document tree. For example, the selector a:visited applies styles only to links that the user has already followed. A CSS pseudo-class is a keyword added to a selector that specifies a special state of the selected element(s) A CSS pseudo-element is used to style specified parts of an element. For example, it can be used to: Style the first letter, or line, of an element Insert content before, or after, the content of an element
