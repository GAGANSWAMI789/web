Control statements-	
	a) Sequence-execution of statements in sequence (step by step)
	
	b) Decision- To execute a set of statements depends upon a particular condition
	
1)if else
			syntax of If-Else
			if (condition)
			{
				..... true case
			}
			else
			{
			....false case
			}
		2)switch case
		switch(var)
		{
			case value:
			case value:
			default:
		}

	c) Iteration-Loop- to execute a set of statements untill any given number of times or any given condition is not satisfied
