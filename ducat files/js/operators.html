Types of operators-
	1)Arithmetic Operator- +, -, *, /, %(ramainder after division), **(power)
	2)Logical operator- &&(and)  ||(or)  !(not)
	
	3)JavaScript Assignment Operators

Operator			Description					Example
=				Assign 10+10 = 20
+=				Add and assign	var a=10; a+=20; Now a = 30
-=	Subtract and assign	var a=20; a-=10; Now a = 10
*=	Multiply and assign	var a=10; a*=20; Now a = 200
/=	Divide and assign	var a=10; a/=2; Now a = 5
%=	Modulus(remainder) and assign	var a=12; a%=2; Now a = 2

4)Comparision Operators- ==     ===    >   <  >=  <= 
	 !=	Not equal to	10!=20 = true
	!==	Not Identical	20!==20 = false
5)Post Increment/Post decrement   eg. a=10  a++   a--
6)Pre Decrement/Pre Increment    eg. a=10   ++a   --a
7) Ternary operator-
		
	(?:)	Conditional Ternary operator-one line statement- returns value based on the condition. It is like if-else.

	(condition)?true :false
