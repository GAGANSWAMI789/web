JS is object based script language.
Every object has
1)Property
2)Method(function)

eg. document- owner object
Property of document object= innerHTML  etc.
Method of document object=getElementById(), write()  etc.

Function- A set of statements to do a particular task is known as function, which can be called by its
 name. function to be executed must be invoked(called). function can be invoked multiple times.
Function types-
1)Library Function(readymade function)-
	a) Math Methods(Number)
	b) String Methods(Text)
	c) Array Methods(Array)
	d) Date Methods
		etc.



2)User Defined Functions(UDF)

Date Methods(Function)

get and set date methods

get methods- to display any information from date methods set methods - to change any information from date methods
